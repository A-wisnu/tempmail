## 🚀 Quick Start

1. **Clone the repository**
```bash
git clone https://github.com/A-wisnu/tempmail.git
```

2. **Install dependencies**
```bash
cd tempmail
npm install
```

3. **Run development server**
```bash
npm run dev
```

4. **Build for production**
```bash
npm run build
```
